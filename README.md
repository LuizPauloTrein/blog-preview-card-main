# 📝 Blog Preview Card

Projeto desenvolvido como parte de um desafio do Frontend Mentor.

## 📌 Sobre o projeto

Este projeto consiste em um card de preview de blog, focado na construção de layouts modernos utilizando apenas **HTML5** e **CSS3**.

O objetivo foi praticar estruturação semântica, estilização e organização de layout, além de aplicar conceitos de responsividade.

---

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- Flexbox

---

## 🎯 O que eu pratiquei

- Estruturação semântica
- Organização de arquivos CSS
- Uso de variáveis CSS
- Alinhamento com Flexbox
- Responsividade básica
